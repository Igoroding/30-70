# 30/70: Я — бэнд-лидер!

Короткий one-pager проекта AI-ассистента для бенд-лидеров и творческих предпринимателей.

## Как открыть

- Локально: открыть `index.html` или `30-70-onepager.html` в браузере.
- После публикации через GitHub Pages: открыть ссылку репозитория.

## Файлы

- `index.html` — основная страница для GitHub Pages.
- `30-70-onepager.html` — автономная отправляемая версия one-pager.
- `30-70-summary.html` — более подробное резюме идеи.
